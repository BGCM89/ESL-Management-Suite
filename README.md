# ESL Management Suite (Automated Billing & Progress Tracker)

### 🚀 The Mission
Transitioning from a background in **Computer Engineering** to **English Education**, I recognized a significant administrative burden in managing private students. This project automates the "business side" of teaching, allowing for more focus on pedagogy while maintaining data-driven insights into student progress.

### 🛠️ Tech Stack & Integration
*   **Language:** Python
*   **Data Source:** Google Sheets API (used as a relational database)
*   **Time Management:** Pytz (Localization for Itatiba/Brazil timezone)
*   **Automation:** Calendar-based weekday frequency calculation

### 📈 Current Features
*   **Dynamic Billing:** Automatically calculates monthly totals based on the specific calendar days (e.g., counting exactly how many Mondays and Wednesdays occur in June 2026).
*   **Portuguese Localization:** All student-facing communication is automatically translated and localized for the Brazilian market.
*   **Time-Aware Greetings:** Contextual "Bom dia/Boa tarde" based on the current hour in Brazil.
